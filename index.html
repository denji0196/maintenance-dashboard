Dashboard งานประจำวัน - พนักงานซ่อม

ใช้บันทึกงานในตารางเวลาโดยตรง กดที่ช่อง PLAN เพื่อกรอกรายละเอียดงาน และกดที่ ACTUAL เพื่อทำเครื่องหมายว่างานได้ทำจริงแล้ว

วันที่ทำงาน
-

แผนงานเทียบงานจริงรายช่วงเวลา

PLAN: คลิกเพื่อกรอกรายละเอียดและเลือกช่วงเวลาเป็นระยะ / ACTUAL: คลิกเพื่อเปลี่ยนเป็นสีเขียว
สีฟ้า = วางแผนไว้, สีเขียว = ทำจริงแล้ว

คำแนะนำการใช้งาน

1. เปิดไฟล์นี้ในเว็บเบราว์เซอร์ได้ทันทีโดยไม่ต้องติดตั้งโปรแกรมเพิ่ม
2. ข้อมูลจะถูกเก็บอัตโนมัติไว้ในเบราว์เซอร์เครื่องที่ใช้งาน (Local Storage)
3. คลิกช่อง PLAN เพื่อกรอก "เวลา", "Line", "Machine", "ประเภทงาน" และ "รายละเอียดงาน"
4. ติ๊ก Request QC หรือ Request MFG เพื่อแสดง badge บนตารางเวลา
5. คลิกช่อง ACTUAL เพื่อทำเครื่องหมายสีเขียวว่างานช่วงเวลานั้นทำจริงแล้ว
